# website
Corporate website for Murdcashi Inc, built with Django and Next.js.

Murdcashi Inc is the official corporate website of a holding company built with a strict, clean-architecture approach. The platform uses Django for the backend and Next.js for the frontend, emphasizing modularity, clear boundaries, and long-term maintainability. It is designed as a scalable digital foundation, with future-ready support for expansion into fintech, e-commerce, and additional subsidiaries without structural refactors.
